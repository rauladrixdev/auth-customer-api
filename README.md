# Auth API

API REST com autenticação usando JWT.

## Funcionalidades
- Registro de usuário
- Login com token JWT
- Hash de senha com bcrypt

## Tecnologias
- Python
- FastAPI
- SQLAlchemy

## Executar

pip install -r requirements.txt  
uvicorn main:app --reload  

Acesse:
http://127.0.0.1:8000/docs
